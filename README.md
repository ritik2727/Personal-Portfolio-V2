# Ritik Jain — Portfolio

World-class developer portfolio built with **Next.js 14** + **Tailwind CSS** + **Framer Motion**.

## ✨ Features

- 🎨 Custom cursor with smooth follower animation
- ✍️ Typewriter / typing animation hero
- 🌐 WebGL particle canvas background
- 📊 Animated skill bars triggered on scroll
- 🗂️ Interactive tabbed experience timeline
- 💼 Filterable projects grid with hover effects
- 📡 Marquee tech stack ticker
- 🎯 Scroll-reveal animations throughout
- 🌙 Noise texture overlay for depth
- 📱 Fully responsive mobile layout

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

```
src/
  app/
    layout.tsx       # Root layout + metadata
    page.tsx         # Main page composition
    globals.css      # Global styles + animations
  components/
    CustomCursor.tsx # Animated cursor
    Navbar.tsx       # Navigation bar
    Hero.tsx         # Landing section with canvas particles
    About.tsx        # About section with orbital animation
    Skills.tsx       # Animated skill bars + marquee
    Experience.tsx   # Tabbed work history
    Projects.tsx     # Filterable project cards
    Contact.tsx      # Contact section
    Footer.tsx       # Footer
```

## 🎨 Design System

- **Primary color**: `#00ff87` (neon green accent)
- **Secondary**: `#60efff` (cyan)
- **Background**: `#050505` (near-black)
- **Font**: Syne (body) + JetBrains Mono (code)

## 🛠️ Tech Stack

- **Next.js 14** — App Router
- **Tailwind CSS** — Utility-first styling
- **TypeScript** — Type safety
- **Framer Motion** — Animations (installed, ready to enhance)
- **Canvas API** — Particle system in hero
